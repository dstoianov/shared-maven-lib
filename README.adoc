== Requirements

. Git
. Java JDK 11
. Apache Maven v3.6.3 or above - *optional

== Add maven wrapper

```bash
mvn --version
mvn -N io.takari:maven:wrapper
mvn -N io.takari:maven:wrapper -Dmaven=3.8.2
```
